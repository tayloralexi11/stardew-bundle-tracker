# 🌿 Stardew Bundle Tracker

A cute, pastel checklist for tracking your **Stardew Valley Community Center** bundle progress — on desktop or phone.

**Live site:** https://tayloralexi11.github.io/stardew-bundle-tracker/

## Features
- ✅ Check items off by **bundle**, by **season**, a **Cart shopping list**, or a **Today** (season + weather) view — all synced.
- 🖼️ Real item sprites from the Stardew Valley Wiki (bundled locally, works offline).
- 💾 Progress auto-saves in your browser. Backup/Restore to a JSON file to move between devices.
- 🔎 Search · 🔥 tricky-item filter · 🗓️ in-season highlight · 💰 Vault gold tracker.
- 🌙 Dark mode · 🔔 sounds · 🌸 seasonal petals · 🎉 confetti · 🐰 Junimo mascot.
- 📱 Installable PWA — "Add to Home Screen" for a fullscreen, offline app.

## Notes
- Standard (non-randomized) bundles, current as of patch 1.6.x.
- Your progress lives only in your own browser — it is never uploaded.
